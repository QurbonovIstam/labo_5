# Лаба 5 — Agile и Scrum

Проект представляет собой учебную модель простого приложения
для управления задачами. Цель проекта — продемонстрировать
применение Agile-подхода и Scrum-процесса с использованием GitHub Issues.
В рамках лабораторной работы был сформирован product backlog,
спланирован один спринт и смоделирован его ход.

## Sprint Review

В рамках спринта были завершены задачи по регистрации пользователя
и созданию задач. По итогам спринта была доставлена базовая
функциональность управления задачами. Задача по редактированию
задач не была завершена и перенесена на следующий спринт.

## Sprint Retrospective

Хорошо прошло планирование задач и распределение приоритетов.
Сложности возникли с оценкой времени выполнения задач.
В следующем спринте стоит более точно оценивать сложность
и уменьшить количество задач.
